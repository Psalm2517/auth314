Pi Network verification for Discord, Telegram, and beyond.

Auth314 connects Pi Network identity to community platforms. Verify once with Pi Sign-in, get access everywhere.

How it works

  1. Your bot calls /api/verify/init
  2. User clicks the link and signs in with Pi
  3. Auth314 delivers the verified identity to your callback URL

Webhook payload

{
  "platform": "discord",
  "platform_user_id": "...",
  "guild_id": "...",
  "pi_uid": "...",
  "pi_username": "..."
}
